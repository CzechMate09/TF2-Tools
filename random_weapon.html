Random Weapon generator

This generator creates a random weapon with randomized attributes.

It includes most weapons and all known stats available in TF2.

Options:

Match Image with Name: If enabled, the weapon's image will match its name.

Match Weapon Type with Name: If enabled, the weapon type (displayed next to "level") will match the weapon's name.

Class: Filters weapons by the selected class.

Slot: Filters weapons by the selected slot.

Weapon Type: Filters weapons by the selected weapon type.

Weapon: Filters weapons by the selected weapon.

Use Normal Stats: If enabled, the generator will include normal stats.

Use Hidden Stats: If enabled, the generator will include stats that are normally hidden in the game.

Use Unused Stats: If enabled, the generator will include stats that were unused in TF2.

Use Random Weapon Description: If enabled, the generator will assign a random description from a random weapon.

Minimal Stat Value: The minimum value that can be assigned to a stat. (Default: 1)

Maximum Stat Value: The maximum value that can be assigned to a stat. (Default: 100)

Use Numbers Multiple Of: If enabled, stat values will be rounded to multiples of the specified number.

Min Amount of Stats: The minimum number of stats a weapon can have. (Default: 1)

Max Amount of Stats: The maximum number of stats a weapon can have. (Default: 5)

Custom Weapon Level Range: Defines the minimum and maximum possible weapon levels.
